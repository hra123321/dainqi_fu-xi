# v2.2 迭代报告 — 公式渲染修复与动态KaTeX重渲染

## 一、本次修复内容

### 1. 公式渲染修复（核心）
- **问题**: 动态加载的题目和批改结果中包含大量 LaTeX 公式（如 `$I_B$`、`$U_{CE}$`），但 KaTeX 只在页面加载时运行一次，动态内容中的公式未被渲染
- **修复**: 在 `renderQuestions()` 和 `renderResults()` 两个函数中，动态插入 HTML 后立即调用 `renderMathInElement()` 重新渲染公式
- **效果**: 题目内容、选项、批改分析中所有 LaTeX 公式都能正确显示为带角标/分式的立体公式

### 2. renderResults 异步Bug修复
- **问题**: `renderResults()` 函数内部使用了 `await fetch()` 但函数声明为普通函数（非 async），导致 JavaScript 运行时错误
- **修复**: 将 `function renderResults()` 改为 `async function renderResults()`

### 3. 知识点自动展示
- **功能**: 切换学科时自动展示该学科全部知识点列表，提升交互流畅度
